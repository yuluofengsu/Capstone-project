# 社区化图书共享与推荐平台 用例描述表

## 用户侧用例描述

### 表2-1 用户注册用例描述表

| 项目 | 描述 |
|-----|------|
| **用例名称** | 用户注册 |
| **用例参与者** | 新用户 |
| **用例描述** | 新用户通过提交邮箱和密码进行账户注册 |
| **前置条件** | 用户打开系统，处于未登录状态 |
| **基本事件流** | 1. 新用户点击注册按钮进入注册页面。<br>2. 用户输入邮箱、密码等信息。<br>3. 系统验证邮箱格式和密码强度。<br>4. 注册成功，跳转至登录页面。<br>5. 用例结束。 |
| **后置条件** | 用户账户创建成功，可进行登录操作 |
| **规则与约束** | 用户邮箱必须符合格式规范；密码需满足强度要求；相同邮箱不可重复注册 |
